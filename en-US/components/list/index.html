    ##API

    ProList API

    Compared with antd's List, ProList's API design is more like Table, so that the presentation form of data items can be quickly defined by configuration. It also makes it easier to switch between Table and List. In addition, ProList is implemented based on ProTable. Except for Table-related APIs, ProList supports most of ProTable's APIs.
