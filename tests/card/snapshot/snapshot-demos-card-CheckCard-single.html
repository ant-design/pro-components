CheckCard.Group Props 说明：

  • onChange : 选中值改变时的回调函数，参数为选中的 value
  • defaultValue : 默认选中的值
  • value : 受控的选中值
  • multiple : 是否支持多选，默认为 false（单选）
  • children : CheckCard 子组件

CheckCard 在 Group 中的 Props：

  • value : 选项的值，用于标识该选项
  • title : 卡片标题
  • description : 卡片描述
  • disabled : 是否禁用该选项
