ProCard 复杂分割布局 Props 说明：

  • title : 卡片标题
  • extra : 卡片右上角额外内容
  • variant : 卡片变体样式，'outlined' 表示带边框
  • headerBordered : 是否显示头部边框
  • split : 分割方式，根据响应式状态动态调整

Split 分割方式：

  • horizontal : 水平分割，子卡片上下排列
  • vertical : 垂直分割，子卡片左右排列
  • 响应式切换 : 根据屏幕尺寸自动切换分割方式

嵌套布局结构：

  • 外层卡片 : 主要容器，控制整体分割方向
  • 中层卡片 : 中间容器，处理水平分割
  • 内层卡片 : 内层容器，处理垂直分割
  • 叶子卡片 : 最终的内容卡片

响应式布局特点：

  • 断点控制 : 当容器宽度小于 596px 时切换布局
  • 动态分割 : 根据屏幕尺寸自动调整分割方向
  • 实时监听 : 使用 ResizeObserver 实时监听容器大小变化

布局效果：

  • 大屏幕 : 使用垂直分割，充分利用水平空间
  • 小屏幕 : 使用水平分割，适合垂直滚动
  • 嵌套结构 : 支持多层嵌套的分割布局

使用场景：

  • 数据仪表盘 : 构建复杂的数据展示界面
  • 报表系统 : 展示多维度统计报表
  • 监控面板 : 构建系统监控面板
  • 分析页面 : 数据分析和可视化页面

最佳实践：

  • 合理嵌套 : 避免过深的嵌套层级
  • 响应式设计 : 确保在不同屏幕尺寸下都有良好的显示效果
  • 性能优化 : 大量数据时考虑虚拟化或分页
