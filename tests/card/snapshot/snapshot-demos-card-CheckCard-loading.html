CheckCard Loading Props 说明：

  • loading : 是否显示加载状态，布尔值

Loading 状态特点：

  • 骨架屏 : 显示骨架屏效果，模拟内容加载
  • 禁用交互 : 加载状态下卡片不可点击
  • 视觉反馈 : 提供明确的加载状态指示
  • 自动布局 : 保持与正常状态相同的布局结构

使用场景：

  • 数据加载 : 异步获取卡片数据时显示
  • 网络请求 : 等待 API 响应时显示
  • 初始化 : 组件初始化过程中显示
  • 状态切换 : 从一种状态切换到另一种状态时显示

与其他属性的组合：

  • loading + title : 加载状态下仍可显示标题
  • loading + avatar : 头像区域显示骨架屏
  • loading + description : 描述区域显示骨架屏

最佳实践：

  • 合理时机 : 在真正需要加载时使用，避免过度使用
  • 用户体验 : 提供清晰的加载状态反馈
  • 性能考虑 : 避免不必要的加载状态切换
