ProCard Hoverable Props 说明：

  • hoverable : 是否启用悬停效果，布尔值
  • variant : 卡片变体样式，'outlined' 表示带边框
  • style : 卡片样式对象，设置最大宽度
  • children : 卡片内容

Hoverable 特点：

  • 悬停效果 : 鼠标悬停时显示阴影和边框效果
  • 交互反馈 : 提供视觉反馈，增强用户体验
  • 可点击提示 : 暗示卡片可以点击或交互

Variant 变体说明：

  • outlined : 带边框的卡片样式
  • filled : 填充式卡片样式
  • elevated : 提升式卡片样式，带有阴影

悬停效果：

  • 阴影变化 : 悬停时阴影加深或出现
  • 边框变化 : 边框颜色或样式发生变化
  • 背景变化 : 背景色可能发生轻微变化

使用场景：

  • 可点击卡片 : 需要点击交互的卡片
  • 导航卡片 : 作为导航链接的卡片
  • 选择卡片 : 需要选择的卡片
  • 产品卡片 : 产品展示卡片

最佳实践：

  • 交互一致性 : 确保悬停效果与实际的交互行为一致
  • 视觉层次 : 使用悬停效果突出重要的卡片
  • 用户体验 : 提供清晰的交互反馈
