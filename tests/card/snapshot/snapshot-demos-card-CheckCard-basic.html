CheckCard Props 说明：

  • avatar : 卡片头像，可以是图片 URL 或 React 节点
  • title : 卡片标题，可以是字符串或 React 节点
  • description : 卡片描述信息，可以是字符串或 React 节点
  • onChange : 选中状态改变时的回调函数，参数为 checked（布尔值）
  • defaultChecked : 默认是否选中，布尔值
  • onClick : 点击卡片时的回调函数
  • checked : 受控的选中状态，布尔值
  • disabled : 是否禁用，布尔值
  • size : 卡片尺寸，可选值：'default' | 'small' | 'large'
