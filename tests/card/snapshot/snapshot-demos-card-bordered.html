ProCard Props 说明：

  • title : 卡片标题，可以是字符串或 React 节点
  • extra : 卡片右上角操作区域，可以是字符串或 React 节点
  • tooltip : 卡片的提示信息
  • style : 卡片的样式对象
  • variant : 卡片变体样式，'outlined' 表示带边框的卡片样式
  • children : 卡片内容区域

Variant 说明：

  • outlined : 带边框的卡片样式，适合需要明确边界的场景
  • filled : 填充式卡片样式，背景色填充
  • elevated : 提升式卡片样式，带有阴影效果
