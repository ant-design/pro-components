CheckCard.Group Options 列表渲染说明：

  • options : 数据源数组，每个元素会被渲染为一个 CheckCard
  • dataSource : 包含多个选项数据的数组

Option 对象属性：

  • title : 卡片标题，显示文本
  • avatar : 卡片头像，可以是 React 节点
  • description : 卡片描述信息
  • value : 选项值，用于标识和表单绑定

Avatar 组件 Props：

  • size : 头像尺寸，数字值（像素）
  • shape : 头像形状，'square' 表示方形
  • src : 头像图片地址

列表渲染特点：

  • 自动布局 : 根据容器宽度自动排列卡片
  • 响应式 : 在不同屏幕尺寸下自动调整布局
  • 统一样式 : 所有卡片使用相同的样式配置
  • 批量操作 : 支持批量选择和操作

使用场景：

  • 算法选择 : 如示例中的机器学习算法选择
  • 产品展示 : 展示多个产品选项
  • 功能选择 : 选择不同的功能模块
  • 配置选择 : 选择不同的配置选项

最佳实践：

  • 数据结构 : 保持数据结构的统一性和一致性
  • 性能优化 : 大量数据时考虑虚拟滚动
  • 用户体验 : 提供清晰的视觉层次和交互反馈
