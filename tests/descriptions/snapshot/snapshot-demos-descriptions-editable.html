ProDescriptions 可编辑功能 Props 说明：

  • actionRef : 操作引用，用于调用组件方法
  • formProps : 表单属性配置
  • title : 描述列表标题
  • request : 异步请求函数
  • editable : 可编辑配置
  • columns : 列配置数组

FormProps 表单配置：

  • onValuesChange : 表单值变化时的回调函数
  • 其他表单属性 : 支持所有 Form 组件的属性

Column 可编辑配置：

  • copyable : 是否可复制
  • ellipsis : 是否显示省略号
  • editable : 是否可编辑，false 表示禁用编辑
  • formItemProps : 表单项属性配置
  • formItemRender : 自定义表单项渲染函数

ValueType 动态配置：

  • 函数形式 : valueType: (record) => string
  • 条件判断 : 根据数据动态返回 valueType
  • 数据依赖 : 基于当前行数据决定显示类型

FieldProps 字段属性：

  • mode : 选择模式，'multiple' 表示多选
  • 其他属性 : 根据 valueType 传递相应属性

Request 远程数据：

  • 异步函数 : 返回 Promise 对象
  • 数据格式 : [{ label: string, value: any }]
  • 动态选项 : 根据条件动态生成选项

Render 自定义渲染：

  • 参数 : (dom, entity, index, action)
  • dom : 默认渲染的 DOM
  • entity : 当前行数据
  • index : 行索引
  • action : 操作对象，包含 startEditable 等方法

Action 操作方法：

  • startEditable : 开始编辑指定字段
  • cancelEditable : 取消编辑
  • saveEditable : 保存编辑

ValueType 类型：

  • select : 选择框，配合 valueEnum 使用
  • textarea : 文本域
  • date : 日期选择器
  • rate : 评分组件
  • money : 金额格式
  • percent : 百分比格式

使用场景：

  • 内联编辑 : 直接在列表中编辑数据
  • 条件编辑 : 根据条件控制编辑状态
  • 动态表单 : 根据数据动态生成表单
  • 交互操作 : 提供丰富的交互操作

最佳实践：

  • 编辑控制 : 合理控制哪些字段可编辑
  • 用户体验 : 提供清晰的编辑反馈
  • 数据验证 : 在编辑时进行数据验证
  • 状态管理 : 合理管理编辑状态
