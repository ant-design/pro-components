ProDescriptions Request 请求 Props 说明：

  • actionRef : 操作引用，用于调用组件方法
  • title : 描述列表标题
  • request : 异步请求函数，返回数据
  • extra : 右上角额外内容

Request 请求配置：

  • 异步函数 : 返回 Promise 对象
  • 数据格式 : { success: boolean, data: object }
  • 自动加载 : 组件挂载时自动请求数据
  • 错误处理 : 自动处理请求错误

ProDescriptions.Item 配置：

  • dataIndex : 数据字段名，自动绑定到 request 返回的数据
  • label : 标签文本
  • valueType : 值类型，决定如何渲染数据
  • children : 自定义内容，如操作按钮

ActionRef 操作方法：

  • reload : 重新加载数据
  • reset : 重置数据
  • clearSelected : 清除选中状态

ValueType 类型：

  • date : 日期格式，自动格式化显示
  • money : 金额格式，自动添加货币符号
  • option : 操作按钮，支持自定义组件

Extra 额外内容：

  • React 节点 : 支持任何 React 组件
  • 操作按钮 : 常用于添加操作按钮
  • 样式控制 : 可以自定义样式和交互

数据绑定特点：

  • 自动绑定 : 通过 dataIndex 自动绑定数据
  • 类型安全 : 提供完整的类型检查
  • 响应式更新 : 数据变化时自动更新显示

使用场景：

  • 数据展示 : 从 API 获取数据并展示
  • 动态刷新 : 通过按钮刷新数据
  • 操作控制 : 通过 actionRef 控制组件行为
  • 用户交互 : 提供用户操作界面

最佳实践：

  • 错误处理 : 在 request 中处理请求错误
  • 加载状态 : 提供加载状态反馈
  • 数据验证 : 验证返回数据的格式
  • 用户体验 : 提供清晰的操作反馈
