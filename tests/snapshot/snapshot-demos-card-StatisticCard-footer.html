StatisticCard Footer Props 说明：

  • title : 卡片标题
  • extra : 卡片右上角操作区域
  • statistic : 主要统计信息配置
  • chart : 图表区域
  • footer : 底部区域，可以是 React 节点
  • style : 卡片样式对象

Footer 区域特点：

  • 额外信息 : 在卡片底部展示额外的统计信息
  • 布局灵活 : 可以包含多个 Statistic 组件
  • 视觉层次 : 与主要统计信息形成层次对比

Statistic 在 Footer 中的配置：

  • value : 统计数值
  • title : 统计项标题
  • suffix : 数值后缀，如单位
  • layout : 布局方式，'horizontal' 表示水平布局

主要统计信息配置：

  • value : 主要统计数值
  • suffix : 数值后缀
  • description : 统计描述，可以是 Statistic 组件

布局效果：

  • 主要信息 : 在卡片中央突出显示主要统计信息
  • 图表展示 : 在主要信息下方显示图表
  • 底部信息 : 在卡片底部显示次要统计信息

使用场景：

  • 评分展示 : 展示评分和相关的详细数据
  • 数据对比 : 在底部展示对比数据
  • 详细信息 : 在底部展示补充信息
  • 多维度展示 : 展示多个维度的统计数据

最佳实践：

  • 信息层次 : 确保主要信息和次要信息的层次清晰
  • 数据关联 : footer 中的数据应该与主要信息相关
  • 视觉平衡 : 合理分配各部分的视觉空间
