ProCard Headless Props 说明：

  • style : 卡片样式对象，设置最大宽度
  • children : 卡片内容

Headless 特点：

  • 无标题 : 不设置 title 属性，卡片没有标题区域
  • 简洁设计 : 只显示内容区域，界面更加简洁
  • 灵活布局 : 适合作为容器组件使用

样式配置：

  • maxWidth : 设置最大宽度，避免在大屏幕上过宽
  • 响应式 : 在小屏幕上会自动调整宽度

与其他类型的区别：

  • 有标题卡片 : 设置 title 属性，显示标题区域
  • 无标题卡片 : 不设置 title 属性，只显示内容
  • 分割卡片 : 使用 split 属性进行分割布局

使用场景：

  • 内容容器 : 作为内容的容器组件
  • 布局组件 : 用于页面布局和内容分组
  • 信息展示 : 展示简单的信息内容
  • 表单容器 : 作为表单的容器组件

最佳实践：

  • 内容清晰 : 确保内容本身能够表达其用途
  • 样式合理 : 根据内容设置合适的样式
  • 响应式设计 : 考虑在不同屏幕尺寸下的显示效果
