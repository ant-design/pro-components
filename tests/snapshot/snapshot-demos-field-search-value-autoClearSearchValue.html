ProFormSelect 搜索值自动清除 Props 说明：

  • name : 字段名
  • placeholder : 占位符文本
  • allowClear : 是否允许清除
  • width : 控件宽度
  • secondary : 是否使用次要样式
  • mode : 选择模式，'tags' 表示标签模式
  • options : 选项数组
  • fieldProps : 字段属性配置

fieldProps 配置：

  • fieldNames : 字段名映射配置
  • showSearch : 是否显示搜索功能
  • autoClearSearchValue : 是否自动清除搜索值

fieldNames 配置：

  • value : 值字段名
  • label : 标签字段名
  • options : 选项字段名
  • 字段映射 : 将自定义字段名映射为标准字段名

选项配置：

  • v : 选项值
  • l : 选项标签
  • 自定义字段 : 使用自定义字段名结构

搜索功能特点：

  • 搜索输入 : 支持输入关键字进行搜索
  • 自动清除 : 选择后自动清除搜索框的值
  • 标签模式 : 支持多选标签模式
  • 字段映射 : 支持自定义字段名映射

布局特点：

  • 固定宽度 : 使用固定宽度 330px
  • 次要样式 : 使用次要样式显示
  • 响应式设计 : 自动适配不同屏幕尺寸
  • 清晰布局 : 提供清晰的输入和选择界面

使用场景：

  • 多选搜索 : 需要多选和搜索功能的场景
  • 自定义字段 : 使用自定义字段名结构的场景
  • 自动清除 : 需要选择后自动清除搜索值的场景
  • 标签选择 : 需要标签形式多选的场景

最佳实践：

  • 搜索配置 : 根据需求配置搜索功能
