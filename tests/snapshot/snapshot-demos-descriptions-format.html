ProDescriptions 格式化配置 Props 说明：

  • column : 列数，设置为 2 表示每行显示 2 列
  • title : 描述列表标题
  • tooltip : 标题提示信息

ProDescriptions.Item 配置：

  • label : 标签文本
  • valueType : 值类型，决定如何渲染数据
  • fieldProps : 字段属性配置
  • children : 数据内容

FieldProps 格式化配置：

  • format : 格式化字符串，定义显示格式
  • 自定义格式 : 支持 dayjs 的所有格式化选项

ValueType 日期时间类型：

  • date : 日期格式，显示年月日
  • dateTime : 日期时间格式，显示年月日时分秒
  • dateTimeRange : 日期时间区间格式
  • time : 时间格式，显示时分秒
  • fromNow : 相对时间格式，显示多久之前

格式化字符串说明：

  • YYYY : 四位年份
  • MM : 两位月份
  • DD : 两位日期
  • HH : 24小时制小时
  • mm : 分钟
  • ss : 秒

数据格式：

  • 时间戳 : 使用 dayjs().valueOf() 获取时间戳
  • 数组格式 : 日期区间使用数组 [start, end]
  • 相对时间 : fromNow 类型支持相对时间计算

Dayjs 操作：

  • add : 添加时间，如 add(-1, 'month') 表示一个月前
  • valueOf : 获取时间戳
  • 格式化 : 通过 format 方法格式化时间

使用场景：

  • 日期展示 : 展示各种格式的日期时间
  • 时间区间 : 展示时间范围
  • 相对时间 : 展示相对时间，如"一个月前"
  • 格式化定制 : 根据需求定制显示格式

最佳实践：

  • 格式统一 : 在项目中保持日期格式的一致性
  • 用户友好 : 选择用户容易理解的格式
  • 国际化 : 考虑不同地区的日期格式习惯
  • 性能考虑 : 避免频繁的时间格式化操作
