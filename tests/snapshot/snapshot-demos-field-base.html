ProField 基础用法 Props 说明：

  • text : 显示的文本内容
  • mode : 显示模式，'read' 表示只读，'edit' 表示编辑
  • valueType : 值类型，决定如何渲染内容
  • plain : 是否使用简约模式
  • valueEnum : 枚举值配置，用于选择框等组件

Radio.Group 配置：

  • onChange : 值变化回调函数
  • value : 当前选中的值
  • 模式切换 : 用于切换 ProField 的显示模式

Switch 配置：

  • checked : 是否选中
  • onChange : 值变化回调函数
  • 简约模式 : 控制是否使用简约模式

Descriptions 配置：

  • column : 列数配置
  • 内容展示 : 用于展示各种 ProField 组件

支持的 valueType 类型：

  • text : 文本类型
  • avatar : 头像类型
  • image : 图片类型
  • money : 金额类型
  • color : 颜色类型
  • digit : 数字类型
  • digitRange : 数字范围类型
  • second : 秒格式化类型
  • percent : 百分比类型
  • rate : 评分类型
  • select : 选择框类型
  • date : 日期类型
  • dateRange : 日期区间类型
  • dateTimeRange : 日期时间区间类型
  • time : 时间类型
  • timeRange : 时间区间类型
  • password : 密码类型
  • code : 代码块类型
  • jsonCode : JSON 代码块类型

valueEnum 配置：

  • text : 显示文本
  • status : 状态类型
  • disabled : 是否禁用
  • 枚举映射 : 将值映射为显示文本和状态

显示模式特点：

  • 只读模式 : 仅显示内容，不可编辑
  • 编辑模式 : 可以编辑内容
  • 简约模式 : 使用简约的显示样式
  • 动态切换 : 支持动态切换显示模式

布局特点：

  • 控制区域 : 顶部显示模式切换控制
  • 展示区域 : 使用 Descriptions 展示各种字段类型
  • 响应式设计 : 自动适配不同屏幕尺寸
  • 网格布局 : 使用网格布局展示多个字段

使用场景：

  • 字段展示 : 展示各种类型的字段内容
  • 模式切换 : 测试只读和编辑模式的切换
  • 样式测试 : 测试简约模式和普通模式的样式差异
  • 功能演示 : 演示各种字段类型的功能

最佳实践：

  • 类型选择 : 根据数据类型选择合适的 valueType
  • 枚举配置 : 合理配置 valueEnum 映射关系
  • 模式设计 : 合理设计只读和编辑模式
  • 用户体验 : 提供清晰的模式切换和内容展示
