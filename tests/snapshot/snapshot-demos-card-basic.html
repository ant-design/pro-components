ProCard Props 说明：

  • title : 卡片标题，可以是字符串或 React 节点
  • variant : 卡片变体样式，可选值：'outlined' | 'filled' | 'elevated'
  • extra : 卡片右上角操作区域，可以是字符串或 React 节点
  • tooltip : 卡片的提示信息
  • style : 卡片的样式对象
  • boxShadow : 是否显示阴影效果，布尔值
  • size : 卡片尺寸，可选值：'default' | 'small'
  • children : 卡片内容区域
