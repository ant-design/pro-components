StatisticCard Horizontal Left Layout Props 说明：

  • chartPlacement : 图表位置，'left' 表示图表显示在左侧
  • statistic : 统计信息配置对象
  • style : 卡片样式对象，使用 maxWidth 控制最大宽度
  • chart : 图表区域

Statistic 配置：

  • title : 统计项标题
  • value : 统计数值
  • precision : 数值精度，控制小数位数
  • suffix : 数值后缀，如单位
  • description : 统计描述，可以包含多个 Statistic 组件

嵌套 Statistic 组件：

  • title : 子统计项标题
  • value : 子统计项数值
  • trend : 趋势指示，'up' 或 'down'

ChartPlacement="left" 特点：

  • 左侧图表 : 图表显示在左侧，统计信息显示在右侧
  • 强调图表 : 左侧的图表更加突出
  • 视觉引导 : 从左到右的阅读习惯

样式配置：

  • maxWidth : 设置最大宽度，避免在大屏幕上过宽
  • 响应式 : 在小屏幕上会自动调整布局

布局效果：

  • 左侧图表 : 可视化图表展示
  • 右侧信息 : 主要统计信息和描述信息
  • 水平布局 : 充分利用水平空间

使用场景：

  • 图表展示 : 强调图表的重要性
  • 数据可视化 : 突出数据可视化效果
  • 趋势分析 : 结合图表展示数据趋势
  • 仪表盘 : 构建数据仪表盘

最佳实践：

  • 图表质量 : 确保左侧图表清晰易懂
  • 信息层次 : 右侧信息应该与图表相关
  • 空间平衡 : 合理分配图表和信息的空间
