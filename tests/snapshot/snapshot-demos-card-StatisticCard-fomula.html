StatisticCard Operation Props 说明：

  • children : 操作符内容，可以是数学符号如 '='、'+'、'-'、'×'、'÷' 等

Operation 组件特点：

  • 数学公式 : 用于在统计卡片之间显示数学运算符号
  • 公式展示 : 展示数据之间的数学关系
  • 视觉连接 : 通过操作符连接多个统计卡片

StatisticCard 配置：

  • statistic.title : 统计项标题
  • statistic.value : 统计数值

公式展示效果：

  • 等号 : 表示总数等于各部分之和
  • 加号 : 表示各部分相加
  • 减号 : 表示减法运算
  • 乘除号 : 表示乘除运算

使用场景：

  • 数据分解 : 展示总数与各组成部分的关系
  • 公式计算 : 展示数学公式和数据计算过程
  • 数据验证 : 验证数据之间的逻辑关系
  • 教学展示 : 用于数据分析和教学场景

最佳实践：

  • 公式清晰 : 确保公式逻辑清晰易懂
  • 数据准确 : 确保公式中的数值计算正确
  • 视觉平衡 : 合理分配各部分的视觉权重
