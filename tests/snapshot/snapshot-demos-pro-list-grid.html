Grid 布局配置
ProList 使用 CSS Grid 实现网格布局，支持自定义列数和间距。通过 grid 属性配置，可以实现响应式的卡片网格布局。
配置面板
列数 (column): 2
1 2 3 4 5 6
间距 (gutter): 8px
0 8 16 24 32 48

幽灵模式:
操作按钮:
代码示例 
                <ProList
  grid={{ gutter: 8, column: 2 }}
  itemCardProps={{ bordered: true, ghost: false }}
  showActions="hover"
  ...
/>
