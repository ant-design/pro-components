StatisticCard Trend Props 说明：

  • title : 统计项标题
  • value : 统计数值
  • trend : 趋势指示，可选值：'up' | 'down' | undefined

Trend 属性说明：

  • up : 上升趋势，显示绿色上升箭头
  • down : 下降趋势，显示红色下降箭头
  • undefined : 无趋势，不显示箭头

趋势显示特点：

  • 颜色区分 : 上升趋势为绿色，下降趋势为红色
  • 箭头指示 : 使用箭头图标直观显示趋势方向
  • 数值对比 : 结合数值变化展示趋势信息

使用场景：

  • 数据对比 : 展示同比、环比等数据变化趋势
  • 指标监控 : 监控关键业务指标的变化趋势
  • 报表分析 : 在报表中展示数据趋势分析
