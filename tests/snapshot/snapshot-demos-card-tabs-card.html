ProCard Tabs Card 类型 Props 说明：

  • tabs.type : 标签页类型，'card' 表示卡片式标签页
  • tabs.items : 标签页配置数组

Tab Item 配置：

  • key : 标签页的唯一标识
  • label : 标签页标题
  • children : 标签页内容

Card 类型特点：

  • 卡片样式 : 标签页以卡片形式显示
  • 视觉分离 : 每个标签页都有独立的卡片边框
  • 突出显示 : 当前选中的标签页更加突出

与其他类型的区别：

  • line 类型 : 默认类型，使用下划线标识当前标签
  • card 类型 : 使用卡片样式，视觉分离更明显
  • editable-card 类型 : 可编辑的卡片式标签页

使用场景：

  • 产品展示 : 展示不同产品的详细信息
  • 功能模块 : 展示不同的功能模块
  • 内容分类 : 对内容进行分类展示
  • 设置页面 : 不同设置选项的展示

最佳实践：

  • 标签清晰 : 确保标签标题清晰易懂
  • 内容相关 : 确保标签页内容与标题相关
  • 数量控制 : 避免过多的标签页影响用户体验
