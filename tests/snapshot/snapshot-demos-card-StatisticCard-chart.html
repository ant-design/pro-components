StatisticCard Chart Props 说明：

  • title : 卡片标题
  • tooltip : 提示信息，鼠标悬停时显示
  • style : 卡片样式对象
  • extra : 卡片右上角操作区域
  • chart : 图表区域，可以是图片或图表组件

Chart 属性说明：

  • 图片图表 : 使用 img 标签显示静态图表
  • 动态图表 : 可以集成 ECharts、AntV 等图表库
  • 响应式 : 图表会自动适配容器宽度

Img 标签属性：

  • src : 图片源地址
  • alt : 图片替代文本，用于无障碍访问
  • width : 图片宽度，'100%' 表示占满容器宽度

图表类型支持：

  • 静态图片 : SVG、PNG、JPG 等格式的图表图片
  • ECharts : 可以集成 ECharts 动态图表
  • AntV : 可以集成 AntV 图表库
  • 自定义组件 : 可以传入自定义的图表组件

使用场景：

  • 数据可视化 : 展示各类统计图表
  • 趋势分析 : 展示数据趋势和变化
  • 对比分析 : 展示数据对比和分布
