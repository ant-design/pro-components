ProCard Loading Props 说明：

  • loading : 加载状态，可以是布尔值或 React 节点
  • title : 卡片标题
  • extra : 卡片右上角额外内容
  • layout : 布局方式，'center' 表示居中对齐
  • style : 卡片样式对象
  • children : 卡片内容

Loading 使用方式：

  • 布尔值 : loading= 使用默认的骨架屏效果
  • React 节点 : loading={
    Loading
    } 使用自定义加载内容

默认 Loading 效果：

  • 骨架屏 : 显示骨架屏效果，模拟内容加载
  • 禁用交互 : 加载状态下卡片不可交互
  • 视觉反馈 : 提供明确的加载状态指示

自定义 Loading：

  • 自定义内容 : 可以传入任何 React 节点作为加载内容
  • 样式控制 : 完全控制加载内容的样式和布局
  • 品牌一致性 : 使用符合品牌风格的加载内容

使用场景：

  • 数据加载 : 异步获取数据时显示
  • 网络请求 : 等待 API 响应时显示
  • 初始化 : 组件初始化过程中显示
  • 状态切换 : 从一种状态切换到另一种状态时显示

最佳实践：

  • 合理时机 : 在真正需要加载时使用，避免过度使用
  • 用户体验 : 提供清晰的加载状态反馈
  • 性能考虑 : 避免不必要的加载状态切换
  • 自定义设计 : 根据业务需求设计合适的加载内容
