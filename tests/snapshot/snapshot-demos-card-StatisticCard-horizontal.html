StatisticCard Horizontal Layout Props 说明：

  • chartPlacement : 图表位置，'right' 表示图表显示在右侧
  • statistic : 统计信息配置对象
  • style : 卡片样式对象
  • chart : 图表区域

Statistic 配置：

  • title : 统计项标题
  • value : 统计数值
  • precision : 数值精度，控制小数位数
  • suffix : 数值后缀，如单位
  • description : 统计描述，可以包含多个 Statistic 组件

嵌套 Statistic 组件：

  • title : 子统计项标题
  • value : 子统计项数值
  • trend : 趋势指示，'up' 或 'down'

ChartPlacement="right" 特点：

  • 右侧图表 : 图表显示在右侧，统计信息显示在左侧
  • 强调数据 : 左侧的统计信息更加突出
  • 空间利用 : 充分利用水平空间

布局效果：

  • 左侧信息 : 主要统计信息和描述信息
  • 右侧图表 : 可视化图表展示
  • 水平布局 : 充分利用水平空间

使用场景：

  • 数据展示 : 展示主要数据指标
  • 趋势分析 : 结合图表展示数据趋势
  • 对比分析 : 展示多个时间段的对比数据
  • 仪表盘 : 构建数据仪表盘

最佳实践：

  • 信息层次 : 确保主要信息和次要信息的层次清晰
  • 图表选择 : 根据数据类型选择合适的图表
  • 空间平衡 : 合理分配统计信息和图表的空间
