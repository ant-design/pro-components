CheckCard Size Props 说明：

  • size : 卡片尺寸，可选值：'large' | 'default' | 'small'
  • title : 卡片标题
  • description : 卡片描述信息

Size 尺寸说明：

  • large : 大尺寸，适合重要内容的展示
  • default : 默认尺寸，最常用的尺寸
  • small : 小尺寸，适合紧凑布局

动态切换：

  • Radio.Group : 使用单选按钮组来动态切换 size 属性
  • useState : 使用 React Hook 管理 size 状态
